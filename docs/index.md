# Introduction

This is the official documentation for Mayuresh's Custom ROM builds. 

Here you will find installation instructions, download links, and troubleshooting guides for all officially supported devices.

## Requirements

Before flashing any builds, ensure you meet the following requirements:

- **Platform Tools**: Install the latest [ADB & Fastboot tools](https://developer.android.com/studio/releases/platform-tools).
- **Bootloader**: Your device bootloader must be unlocked.
- **Backups**: Backup all critical data on your device. Formatting data is strictly required.
- **Kernels**: Custom kernels are not supported. Do not flash them or report bugs if you do.
- **Recovery**: Use only the recommended recovery specified in the installation guide for your device.

## Community & Support

Support is provided via our Telegram group. If you encounter bugs, please provide a full `logcat` when reporting issues.
